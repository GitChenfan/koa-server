## koa 后端项目模版
### 基于以下组件
* node 
* koa2
* json web token 认证
* mysql 数据库

### 安装
```bash
npm install
```

### 运行
```bash
npm start          # develop front end
npm run build      # build front end
```

### 配置文件
* config/app.js     -- app config
* config/log4js.js  -- log4js config
* process.json      -- pm2 config
